## This iteration

This version consist out of  vanilla, client only markdown to html converter. Which totally is not developed fully developt at the moment but at least hasr Styles H1, H2, H3 and P. 

Should you not know the meaning of the "Symbols", I will explain it to you. The H in H1 and etc. stands for header, headtext, I am not so sure. You can imagine it as an class which attributes I can change through CSS or JS. CSS being the way to style HTML documents. That means changing color, font weight and font size. I can apply this attributes on all H1 either in a hierarchy, in a new class or even with an ID.

Should I ever have Code snippet support, I will past it in.
