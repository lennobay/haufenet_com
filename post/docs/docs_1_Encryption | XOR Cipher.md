# The XOR Cipher


## Basic Function
A Word can be encrypted by another key in binary and simple addition:

    1 + 1 = 0
    0 + 1 = 1
    0 + 0 = 0

## Example
The word Wiki (pulled as an example from Wikipedia):

+    01010111 01101001 01101011 01101001
+   11110011 11110011 11110011 11110011

=   10100100 10011010 1001100 10011010 (encrypted Text) 

Keyword: 11110011
